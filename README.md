# DeepRitzMethod
This project aims to compute steady states of the Cahn–Hilliard equation under periodic boundary conditions. GL_1dim_trivial.py and GL_2dim_trivial.py is using normal deep ritz method to solve this question in 1d and 2d respectively. GL_1dim_nontrivial.py, GL_2dim_trivial.py and GL_3dim_trivial.py adopt Fourier feature mapping combined with the deep Ritz method to find the non-trivial solutions in 1d, 2d and 3d respectively.
