# DeepRitzMethod
GL_1dim_normal.py and GL_2dim_normal.py is using normal deep ritz method to solve this question in 1d and 2d respectively. GL_1dim.py, GL_2dim_final.py and GL_3dim_final.py adopt Fourier feature mapping combined with the deep Ritz method to find the non-trivial solutions in 1d, 2d and 3d respectively.
