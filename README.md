# DeepRitzMethod
This project focuses on computing steady-state solutions of the Cahn–Hilliard equation under periodic boundary conditions.

## Structure

- `GL_1dim_trivial.py` and `GL_2dim_trivial.py` implement the standard Deep Ritz method to obtain trivial solutions in one and two dimensions, respectively.  
- `GL_1dim_nontrivial.py`, `GL_2dim_nontrivial.py`, and `GL_3dim_nontrivial.py` incorporate Fourier feature mapping into the Deep Ritz framework to capture non-trivial solutions in one, two, and three dimensions.  
- `data.zip` contains datasets for both trivial and non-trivial solutions.

## Notes

This structure provides a clear comparison between baseline and enhanced approaches across different dimensions.
